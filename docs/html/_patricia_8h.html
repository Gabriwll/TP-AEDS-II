Cria um novo nó externo (folha) na árvore PATRICIA.

Um nó externo armazena a chave e um vetor indicando em quais arquivos essa chave ocorre.

Parameters
kChave (string) a ser armazenada no nó.
pPonteiro para o ponteiro onde será criado o nó.
N_arquivoÍndice do arquivo onde a chave foi encontrada (1-based).
Returns
